现在的搜索已经不是只过滤首页已有卡片，而是会读取本地生成的 `search.json`，在浏览器里完成全文匹配。

这篇文章现在放在 `source/_posts/guides/search/local-search-notes.md`，方便和同主题的内容一起归档。

## 这意味着什么

- 搜索范围会覆盖标题、摘要、标签和正文全文
- 命中结果会根据标题、标签和正文出现位置综合排序
- 结果卡片会提取正文片段，并对匹配关键词做高亮显示
- 整个过程不依赖第三方搜索服务，部署到 GitHub Pages 也能直接工作

## 可以直接测试的关键词

- 全文搜索
- sticky
- 归档时间轴
- 标签导航
- search.json
